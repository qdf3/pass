## 🎮 طريقة اللعب

- يولّد الكمبيوتر رقماً سرياً مكوّناً من **4 أرقام مختلفة**
- لديك **10 محاولات** لتخمينه
- بعد كل محاولة تحصل على تلميحات:
  - 🟢 **B** = الرقم صحيح وفي مكانه الصحيح
  - 🟡 **M** = الرقم موجود لكن في مكان خاطئ
  - ⬛ **✕** = الرقم غير موجود في الكود

---

## ✨ المميزات

- 🎨 تصميم احترافي بثيم داكن مع ألوان متدرجة
- ⌨️ انتقال تلقائي بين حقول الإدخال
- 📊 إحصائيات لحظية (المحاولات، المتبقي، الانتصارات)
- 🏆 تأثيرات confetti عند الفوز
- 📱 متجاوب مع جميع الشاشات
- ⚡ ملف واحد فقط — لا مكتبات خارجية

## 🚀 كيفية الاستخدام

